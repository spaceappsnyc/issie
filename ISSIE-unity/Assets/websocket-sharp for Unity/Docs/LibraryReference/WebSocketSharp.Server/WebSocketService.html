Exposes the methods and properties for a WebSocket service provided by the WebSocketSharp.Server.HttpServer or WebSocketSharp.Server.WebSocketServer.

Syntax

Remarks

Protocol Property

Gets or sets the subprotocol of the WebSocketSharp.WebSocket used in the current session.

Syntax

public string Protocol { get; set; }

Value

A string that represents the subprotocol of the WebSocketSharp.WebSocket used in the current session if any.

The value to set must be a token defined in RFC 2616.

The default value is string.Empty.

Remarks

Set operation of this property is available before the WebSocket connection has been established.

ValidateCookies Method

Used to validate the HTTP cookies included in the WebSocket connection request.

Syntax

protected virtual bool ValidateCookies (WebSocketSharp.Net.CookieCollection request, WebSocketSharp.Net.CookieCollection response)

Parameters

request
A WebSocketSharp.Net.CookieCollection that contains the cookies to validate.
response
A WebSocketSharp.Net.CookieCollection that receives the cookies to send to the client.

Returns

true if the cookies are valid; otherwise, false. This method returns true as default.

Remarks

This method is called when the WebSocketSharp.WebSocket used in the current session validates the WebSocket connection request.
